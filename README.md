# Описание проекта

Данное приложение было создано в рамках курса Yandex
practicum ["Android-разработчик"](https://practicum.yandex.ru/android-developer/).

Приложение предназначено для поиска вакансий на площадке hh.ru

Посмотреть подробное задание и описание проекта можно [здесь](./docs/ABOUT.md)

## Краткое описание работы над проектом

Над проектом одновременно работало 4 человека в течении 1 месяца. Задание было разбито на 4 спринта по 1 неделе.
В рамках спринта мы совершали созвоны и распределяли задания между собой.
Сложные задачи, "эпики", разбивали на более мелкие, заводили таски
в [github projects](https://github.com/users/goaliepash/projects/1),
назначали исполнителя.
Вся разработка велась в ветке develop, от которой заводились ветки с тасками.
После написания кода pull request предоставлялся остальным членам команды для кросс-ревью.
И, только после положительного заключения всеми членами команды, мёрджился в основную ветку.

В конце спринта код проверялся куратором и вливался в main.

## Используемый в проекте стек

Была применена архитектура single activity и архитектурный паттерн MVVM.
Экраны созданы на fragment'ах.

Ui - xml
Di - Coin (заменен мной на Dagger Hilt уже после окончания курса)
Network - Retrofit
DB - Room
DataStore - SharedPref ???
Для передачи состояния в ui - LiveData

## Рефакторинг после окончания курса

После окончания курса "Android-разработчик" я прошел курс "Мидл Android-разработчик" от того же Yandex practicum,
где познакомился с более расширенным и современным стеком технологий (compose, flow, websocket, многомодульность,
плагины, новые аритектурные паттерны(MVI, TEA), работа с bluetooth и медиа, пуш уведомления и многое другое),
после чего решил отрефакторить данный проект, используя новые знания.

### Что в итоге было проделано:

Что бы не терять оригинальную версию проекта и иметь возможность их сравнить, я использовал Build Variants.
Оригинальную версию назвал - xml, новую - compose.

В основном изменения коснулись слоев ui и presentation.

Ui полностью заменен на compose (крутой инструмент, очень понравился! прим. автора)
Di - coin заменен на hilt
DataStore - SharedPref заменен на DataStore ???
